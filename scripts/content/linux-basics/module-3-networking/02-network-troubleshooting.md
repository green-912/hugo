# Network Troubleshooting Basics

## TH – บทนำ
สรุป workflow การแก้ปัญหา network พื้นฐาน:
1. ดูว่าเครื่องมี IP หรือไม่
2. ping gateway
3. ping internet
4. ตรวจ DNS

## EN – Introduction
Basic troubleshooting workflow:
1. Check if the host has an IP
2. Ping gateway
3. Ping external site
4. Check DNS
