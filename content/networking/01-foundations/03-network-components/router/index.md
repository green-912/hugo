# Router (Layer 3)

## TH  
Router เชื่อมต่อหลายเครือข่ายเข้าด้วยกัน เช่น LAN ↔ Internet  
ฟังก์ชันหลัก:  
- Routing (เลือกเส้นทาง)  
- NAT  
- Forwarding  
- Gateway ของเครือข่าย  

## EN  
A router connects multiple networks and chooses packet paths.  
Main roles:  
- Routing  
- NAT  
- Forwarding  
- Gateway operations
