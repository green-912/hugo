# Switch (Layer 2)

## TH  
Switch คืออุปกรณ์ที่ทำงานที่ Layer 2 (Data Link) ใช้ MAC Address เพื่อส่ง Frame ไปยังปลายทางที่ถูกต้อง  
คุณสมบัติสำคัญ:  
- สร้าง MAC Table  
- ส่งข้อมูลแบบ Unicast/Multicast/Broadcast  
- สนับสนุน VLAN  
- เป็นหัวใจสำคัญของ LAN

## EN  
A switch is a **Layer 2 device** that forwards frames based on MAC addresses.  
Key features:  
- MAC table learning  
- Unicast/Multicast/Broadcast forwarding  
- VLAN support  
- Backbone of local networks
