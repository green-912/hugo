# What is Networking?
พื้นฐานของระบบเครือข่ายคือ “การสื่อสารข้อมูล” ระหว่างสองปลายทาง  
ในรูปแบบที่มีมาตรฐาน (protocol) และส่งข้อมูลแบบมีโครงสร้าง (packet)

---

## TH – คำอธิบายพื้นฐาน

**Networking คือกระบวนการที่ทำให้คอมพิวเตอร์ อุปกรณ์ หรือระบบ สามารถ “คุยกัน” ได้**

ประกอบด้วย:

- ผู้ส่ง (Sender)
- ผู้รับ (Receiver)
- ข้อมูล (Data)
- วิธีการส่ง (Protocol)
- เส้นทาง/สื่อในการส่ง (Medium)

ตัวอย่างเครือข่ายในชีวิตประจำวัน:

- ดู YouTube → คุยกับ Server ผ่าน TCP/IP  
- ส่งไลน์ → คุยผ่าน API + TLS  
- เปิดเว็บ → ใช้ HTTP → TCP → IP → Ethernet  

คำสำคัญ:

- **Protocol** = กฎการสื่อสาร  
- **Packet** = ชิ้นข้อมูล  
- **Address** = ที่อยู่ของอุปกรณ์ เช่น IP / MAC  

---

## EN – Overview

Networking is **the process that allows two or more devices to exchange data**  
using structured rules (protocols) and defined formats (packets).

Key concepts:

- **Sender / Receiver**
- **Protocol** (a set of rules)
- **Packet** (small unit of data)
- **Addressing** (MAC, IP)
- **Medium** (Ethernet, Wi-Fi)

Examples:

- Browsing websites via HTTP over TCP/IP  
- Sending messages over TLS-encrypted channels  
- Streaming videos over UDP or TCP  

Networking = **Controlled communication between systems**
