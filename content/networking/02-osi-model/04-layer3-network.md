## TH: Layer 3 คืออะไร?

จัดการ:
- IP Address  
- Routing  
- การส่งต่อแพ็กเก็ต  

โปรโตคอลหลัก:
- IPv4 / IPv6  
- ICMP  
- OSPF, BGP  

ปัญหาที่พบบ่อย:
- Routing ผิดเส้นทาง  
- IP ซ้ำกัน  
- Subnet overlap  

## EN: Layer 3 – Network

Handles routing, forwarding, IP addressing.  
Common issues: wrong routes, duplicate IPs, ICMP unreachable.
