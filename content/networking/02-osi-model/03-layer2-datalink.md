## TH: Layer 2 ทำหน้าที่อะไร?

จัดการ:
- MAC Address  
- Switching  
- VLAN Tag (802.1Q)  
- ตรวจ Error เบื้องต้นผ่าน FCS  

โปรโตคอลสำคัญ:
- Ethernet  
- ARP  
- VLAN  

ปัญหาจริง:
- STP loop  
- MAC flapping  

## EN: Layer 2 – Data Link

Handles MAC addressing, switching, VLAN tagging, and basic error detection.  
Issues include loops, ARP anomalies, and MAC flooding.
