## TH: OSI Model คืออะไร

OSI Model คือโมเดลอธิบาย “ขั้นตอนการสื่อสารข้อมูล” ที่แบ่งออกเป็น 7 ชั้น  
ข้อดีของการแบ่งเป็นชั้นๆ คือ:
- คุยกันรู้เรื่องว่าอะไรรับผิดชอบตรงไหน
- แก้ปัญหา (troubleshooting) ได้เป็นระบบ ว่าพังที่ชั้นไหน
- ใช้เป็นฐานคิดเวลาออกแบบระบบ

7 ชั้น (แค่ตั้งชื่อให้จำก่อน):
1. Physical  
2. Data Link  
3. Network  
4. Transport  
5. Session  
6. Presentation  
7. Application  

## EN: What is the OSI Model?

The OSI Model is a conceptual model that describes how data communication works by splitting it into 7 layers.  
This layering helps us:
- Use a common language when discussing networking
- Troubleshoot problems layer by layer
- Design and teach networking systems in a structured way
