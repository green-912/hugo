## TH: Layer 7 คืออะไร?

เป็นชั้นที่ผู้ใช้สัมผัส เช่น:
- HTTP / HTTPS  
- DNS  
- SMTP  
- FTP  

Layer ที่ถูกโจมตีบ่อยที่สุดเพราะมีช่องโหว่เยอะที่สุด

## EN: Layer 7 – Application

Provides services to users: HTTP, DNS, SMTP.  
Most attacked layer due to complexity and exposure.
