## TH: Layer 4 ทำอะไร?

จัดการ end-to-end communication  
โปรโตคอล:
- TCP (reliable)  
- UDP (fast)  

ประเด็นสำคัญ:
- TCP handshake (SYN, SYN/ACK, ACK)  
- Port number  

## EN: Layer 4 – Transport

Provides end-to-end connections using TCP/UDP.  
TCP ensures reliability, UDP is fast and connectionless.
